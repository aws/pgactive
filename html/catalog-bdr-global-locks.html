13.9. bdr.bdr_global_locks

bdr.bdr_global_locks is an implementation detail for BDR's global DDL locking mechanism. See DDL replication and Monitoring. Do not modify this table directly.

You are looking for bdr.bdr_locks, the information view, not for this table. See also DDL Locking.

You should never need to access this table.
