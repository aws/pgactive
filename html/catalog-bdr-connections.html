13.2. bdr.bdr_connections

The bdr.bdr_connections table keeps track of the connection strings used for each node to connect to each other node.

Avoid directly modifying bdr.bdr_connections. Use the provided node management functions instead. See Node management functions. It is safe to modify the connection string.

Table 13-2. bdr.bdr_connections Columns
