13.11. bdr.bdr_queued_drops

bdr.bdr_queued_drops is a BDR internal implementation table used for DDL replication. Do not modify this table directly.

Every table/sequence drop operation that's captured and replicated is inserted in this table. Inspecting this table can be useful to determine what schema changes were made when and by whom.
