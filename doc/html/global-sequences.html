10.8. BDR 1.0 global sequences

BDR global sequences provide an easy way for applications to use the database to generate unique synthetic keys in an asynchronous distributed system.

Important: This chapter refers to the global sequences implementation in BDR 2.0 and newer. See the BDR 1.0 documentation and the upgrade guide for details on the quite different global sequences implementation in BDR 1.0.

Warning

Object-relational mappers and applications that hardcode calls to the nextval function may require special configuration to work with BDR 2.0 global sequences.
